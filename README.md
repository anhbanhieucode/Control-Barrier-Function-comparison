# Control-Barrier-Function-comparison

#Introduction:

This repo is a list of some variation of Control Barrier Functions, including:
1. Basic CBF
2. CBF + velocity optimization
3. CBF + velocity optimization + obstacle position prediction
4. CBF + velocity optimization + obstacle position prediction + soft constraints  

# Requirements:
matplotlib, numpy, cvxpy
How to install
```
pip install matplotlib
pip install cvxpy
```
#How to use:
```
python main_4_compare.py
```
